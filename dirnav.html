1. front

use arrow keys to navigate directionally: ↓ bottom, ← left, → right, or use tab/space to navigate in order (should go right, top, left, bottom, front, ..

2. right

↓ bottom, ← front, ↑/tab/space top

3. top

↓ front, ← left, → right, tab/space left

4. left

→ front, ↑ top ↓/tab/space bottom

5. bottom

← left, → right, ↑/tab/space front
